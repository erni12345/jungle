Projet Jungle

Installation des modules
 - Ce projet utilise Python, il est donc necessaire d'avoire Python
 - Lancer le script bash "install_packages.sh" afin d'installer tout les modules automatiquement


Utilisation
- Sur un terminal lancer le script "main.py" avec "python main.py"
- ouvrir le port utilise par Flask sur une page web


Fonctionement du site:

- Le site est un reseaux sociale qui propose plusieurs choses:
    - Creation de compte
    - Login avec un compte
    - Page d'acceuille avec les postes de toutes les personnes qu'on "follow"
    - Chercher (bar du haut) des utilisateur
    - "Follow" des personnes
    - Acces a son compte complet, voir les postes qu'on a fait etc
    - Creer des postes
